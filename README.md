# OpticalFiberShapeScanner
광섬유 형태 출력기
(2015년 1월)

## 주제
광섬유 내의 3개의 센서의 위치 정보를 획득, 스플라인 보간법을 이용해 형태를 출력

## 핵심 기능
> 광섬유 형태 3D 그래프 출력
> 각종 정보 텍스트 출력
> 일시정지, 멈춤 기능
> 각도에 따라 관점 변화
